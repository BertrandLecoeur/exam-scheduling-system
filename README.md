# Exam Scheduling System

Automated university exam scheduling system developed in Python using graph coloring algorithms and room allocation optimization.

## Overview

This project generates optimized university exam timetables while handling multiple academic and logistical constraints:

- Exam conflicts between courses
- Room capacities
- Room availability
- Student distribution
- Session scheduling
- Multi-day timetable generation

The scheduling process is based on the **DSATUR graph coloring algorithm**, commonly used in combinatorial optimization and scheduling problems.

---

## Features

- Automated exam timetable generation
- Graph coloring scheduling algorithm (DSATUR)
- Room assignment optimization
- Conflict detection between subjects
- CSV data management
- Object-oriented architecture
- Tkinter graphical interface
- Multi-session and multi-room support
